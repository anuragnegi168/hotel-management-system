# 🏨 Hotel Management System (Python)

A console-based Hotel Management System built using Python.  
This project simulates real-world hotel operations like room booking, checkout, billing, and management.

---

## 🚀 Features

- ➕ Add new rooms with validation  
- 📋 View all rooms  
- 🛏️ Book rooms with customer details  
- 🔄 Checkout rooms and reset data  
- 🔍 Search rooms by room number  
- ✏️ Update room details (type & price)  
- ❌ Delete rooms from the system  
- 📊 Show available and booked rooms  
- 💰 Generate bill with 18% tax calculation  
- 📈 Hotel summary (total rooms, available, booked, total income)

---

## 🛠️ Tech Stack

- Python (Core Concepts)
- Lists & Dictionaries
- Functions
- Loops & Conditions
- Exception Handling (try-except)
